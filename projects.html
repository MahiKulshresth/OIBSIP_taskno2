My Projects

1. Library Management System

I did this project in my 3rd semester for RDBMS (Relational Database Management System).
The database was made in MySQL. All the backend coding was done in Python 3 language
using Flask framework. Frontend coding was done in HTML and CSS. The LMS could
perform various operations like adding a book, adding a user, calculating dues,
generating overdue reports and many more.

2. TIC TAC TOE

I did this project in my 3rd semester for OOP (Object Oriented Programming).
All the coding was done in Python 3. I used Tkinter library in python
to generate a simple interface. The game gave both the players (namely X and Y) their
turns alternatively and at the end declares the winning player (if any), otherwise
it displays "It's a draw".

3. Real Estate Price Prediction Model

I did this project in my 3rd semester for PBL Lab (Project Basrd Learning Lab).
All the coding for this machine learning project was done in Python 3.
Libraries used were Scikit Learn, Pandas, NumPy and Matplotlib.
The Supervised Leaning Technique was used to train the model using a predifined dataset.
RandomForest Regressor model was used for training. The whole project was divided into
three modules: Data Preprocessing module, Model Training module and model testing and validation module.
